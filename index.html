Java Full Stack — 24 Weeks (Java 17 → 24, Spring Boot)
Weeks 1–4: Java Core & OOP (classes, objects, inheritance, interfaces, exceptions)
Weeks 5–8: Java Advanced (Generics, Streams API, Lambdas, Concurrency, I/O)
Weeks 9–12: JDBC, JPA, Hibernate basics, Database design
Weeks 13–16: Spring Boot Core (controllers, services, repositories), REST API design
Weeks 17–18: Security (Spring Security, JWT), Testing (JUnit, Mockito)
Weeks 19–20: Frontend basics (HTML, CSS, JS) + React basics for integration
Weeks 21–22: Fullstack Integration (connect React with Spring APIs), session & auth flows
Weeks 23–24: Project: Capstone (e-commerce or management app), Dockerize and deploy
Kotlin Full Stack — 20 Weeks (Spring Boot with Kotlin)
Weeks 1–3: Kotlin Basics (syntax, null-safety, coroutines intro)
Weeks 4–6: Kotlin Advanced (coroutines, flows, DSLs, extension functions)
Weeks 7–10: Spring Boot with Kotlin (Ktor or Spring Boot Kotlin starter), REST APIs
Weeks 11–12: Database (JPA with Kotlin, Exposed ORM), testing
Weeks 13–14: Authentication & Security (JWT), async patterns with coroutines
Weeks 15–16: Frontend basics + React (TypeScript optional) integration
Weeks 17–18: Microservices basics & building modular services
Weeks 19–20: Capstone project, containerization and deployment
MERN Stack — 20 Weeks (MongoDB, Express, React, Node)
Weeks 1–2: HTML, CSS, JavaScript fundamentals & ES6 features
Weeks 3–6: React (components, hooks, state, context, routing)
Weeks 7–9: Node.js & Express (server, middleware, REST APIs)
Weeks 10–11: MongoDB basics & Mongoose (schema design, queries)
Weeks 12–13: Authentication & Authorization (JWT, sessions), file uploads
Weeks 14–15: State management & advanced React patterns; testing (Jest)
Weeks 16–17: Deployment (Vercel for frontend, Render/Heroku for backend), CI basics
Weeks 18–20: Capstone project (fullstack SaaS or marketplace)
Python Full Stack (Django) — 18 Weeks
Weeks 1–3: Python fundamentals (data structures, OOP, virtualenv)
Weeks 4–6: Django basics (models, views, templates, admin)
Weeks 7–9: Django REST Framework (serializers, viewsets, routers)
Weeks 10–11: Authentication, permissions, token/JWT flows
Weeks 12–13: Frontend integration (HTML/CSS/JS) and React/Vue optional
Weeks 14–15: Database optimization, migrations, caching (Redis)
Weeks 16–18: Deployment (Gunicorn + Nginx), Dockerize, project & testing
C Full Stack — 16 Weeks
Weeks 1–4: C basics (syntax, pointers, memory management, file I/O)
Weeks 5–7: Data structures & algorithms in C (linked lists, trees, sorting)
Weeks 8–10: Network programming basics (sockets) & simple server implementation
Weeks 11–12: Integrating with a lightweight web framework or CGI approach (building APIs in C) / or using C as backend component
Weeks 13–14: Database connectivity (libpq / MySQL C API) and SQL basics
Weeks 15–16: Project: small web service or microservice component + deployment
C++ Full Stack — 16 Weeks
Weeks 1–3: C++ core & modern C++ features (STL, RAII, smart pointers)
Weeks 4–6: OOP & advanced C++ (templates, move semantics, concurrency basics)
Weeks 7–9: Backend options in C++ (CppRESTSDK, Pistache, CppCMS) and building REST APIs
Weeks 10–11: Database access (ODBC, libpq) and ORM-like approaches
Weeks 12–13: Integrating with frontend (serving JSON APIs) & testing frameworks
Weeks 14–16: Project: performance-focused service + deployment and monitoring
Android Full Stack — 24 Weeks (Kotlin, Jetpack Compose, NDK & Modularization)
Weeks 1–3: Kotlin for Android, Android fundamentals
Weeks 4–7: Jetpack Compose UI, layouts, state, navigation
Weeks 8–10: Android Architecture (ViewModel, LiveData / StateFlow), Room DB
Weeks 11–13: Networking (Retrofit), coroutines, offline support
Weeks 14–16: Modularization (feature modules), clean architecture patterns
Weeks 17–18: NDK basics (C/C++ interop), JNI, native modules
Weeks 19–20: Integrating backend (REST / GraphQL), auth flows
Weeks 21–24: Testing, Google Play publish flow, CI/CD, capstone app

Weeks 1–2: SQL basics (SELECT, INSERT, UPDATE, DELETE)

Weeks 7–8: Stored procedures, triggers, functions, views
Weeks 9–10: Advanced topics (partitioning, replication basics, backups)
Weeks 11–12: Hands-on with MySQL/Postgres/SQL Server, security & projects
